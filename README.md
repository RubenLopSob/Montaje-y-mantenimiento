# Proyecto Intermodular: Montaje y mantenimiento de equipos

Este repositorio contiene 5 archivos, cada uno de los 5 primeros entregables descritos en el enunciado del proyecto intermodular que se nos entregó hace unos meses:

- **El análisis de necesidades de hardware** analiza qué se necesitará para cada dispositivo según el contexto de la empresa.

- **La configuración de equipos** especifica qué componentes o modelos se han elegido en cada caso, acorde a las necesidades de hardware.

- **La explicación del montaje** explica cómo se han de montar los equipos de administración y Linux para clientes paso a paso.

- **El plan de mantenimiento** especifica cómo se mantendrán los equipos y qué medidas se tomarán para que funcionen el mayor tiempo posible y en buen estado.

- **El inventario de equipos** muestra cuántos equipos de cada tipo se han comprado, parte de sus componentes y su precio.
